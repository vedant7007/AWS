# AWS Cloud Club VJIT

Official website for the AWS Cloud Club at Vidya Jyothi Institute of Technology (VJIT), Hyderabad.

## Tech Stack

- **Framework:** Next.js (App Router) with React & TypeScript
- **Styling:** Tailwind CSS v4 with custom design tokens
- **Animations:** GSAP (ScrollTrigger, SplitText), Framer Motion
- **Smooth Scroll:** Lenis
- **Icons:** Lucide React

## Pages

- **Home** — Hero, About, Benefits, Achievements, Testimonials, FAQ, CTA
- **Events** — Upcoming & past events with filtering
- **Team** — Core team members with holographic profile cards
- **Gallery** — 3D dome gallery of club moments
- **Contact** — Get in touch form
- **Feedback** — Anonymous feedback submission

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Deployed on [Vercel](https://vercel.com).
